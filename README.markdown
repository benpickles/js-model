# js-model

js-model is a library that allows you to work with models in your JavaScript.

## In brief

Declare a model class:

    var Todo = Model("todo")

Now create and manipulate model instances:

    var todo = new Todo({ text: "do it" })
    todo.attr("when", "now")
    todo.save()
